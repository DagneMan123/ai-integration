User made payment but credits were not being added to wallet.
Balance remained 0 even after successful payment.

ROOT CAUSE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Frontend was NOT sending bundleId when initializing payment.

BEFORE (Bug):
  Frontend sends: { amount: 5, creditAmount: 1 }
    ↓
  Backend receives: bundleId = null, creditAmount = 1
    ↓
  Payment created with: creditAmount: 1
    ↓
  Wallet updated: +1 credit (instead of +5)
    ↓
  User sees: 1 credit ❌ WRONG

AFTER (Fixed):
  Frontend sends: { bundleId: 1, amount: 25, creditAmount: 5 }
    ↓
  Backend receives: bundleId = 1, creditAmount = 5
    ↓
  Payment created with: creditAmount: 5
    ↓
  Wallet updated: +5 credits
    ↓
  User sees: 5 credits ✅ CORRECT
