## How to Use

### 1. Page Loads
- System checks start automatically
- You see loading spinners
- Wait for all checks to complete

### 2. View Results
- Green checkmarks = Ready
- Amber warnings = May need attention
- Red errors = Must fix before proceeding

### 3. Fix Issues
- Click on any failed check
- Read the troubleshooting steps
- Follow the instructions
- Click "Re-check" to verify fix

### 4. Start Interview
- All checks must pass (green)
- "Start Interview" button becomes active
- Click to proceed

## Common Issues & Fixes

### Camera Not Working
1. Check camera is connected
2. Allow camera permission in browser
3. Disable camera blocking software
4. Try different browser

### Microphone Not Working
1. Check microphone is connected
2. Allow microphone permission in browser
3. Check volume is not muted
4. Try different browser

### No Internet
1. Check WiFi connection
2. Restart router
3. Move closer to router
4. Use wired connection

### Browser Not Supported
1. Update browser to latest version
2. Use Chrome, Firefox, Safari, or Edge
3. Disable browser extensions
4. Clear browser cache

## Browser Recommendations
