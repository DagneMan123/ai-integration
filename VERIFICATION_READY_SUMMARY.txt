⚠️ IMPORTANT NOTES

1. PostgreSQL MUST be running before starting backend
   Command: net start postgresql-x64-16

2. Backend MUST be running before testing API endpoints
   Command: cd server && npm run dev

3. Frontend MUST be running before testing dashboards
   Command: cd client && npm start

4. Database MUST have data before testing dashboards
   Create test users and jobs during verification

5. AI features work in two modes:
   ├─ Live mode: When OpenAI quota available
   └─ Fallback mode: When quota exceeded (system still works)

6. All data is stored in PostgreSQL database
   Check with: npx prisma studio

7. API endpoints require authentication (JWT token)
   Obtained after login

8. Each dashboard has different permissions
   ├─ Candidate: Can only see own data
   ├─ Employer: Can see own jobs and candidates
   └─ Admin: Can see all data
