ROOT CAUSE:
- Chapa doesn't pass tx_ref back in return URL
- Even though we added it to return_url, Chapa ignores it

SOLUTION:
✅ Use localStorage to store tx_ref before redirect
✅ Retrieve tx_ref from localStorage after Chapa redirects back
✅ This is the ONLY reliable way to get tx_ref

HOW IT WORKS:

1. User clicks "Subscribe Now"
   ↓
2. Backend returns tx_ref and checkout URL
   ↓
3. Frontend stores tx_ref in localStorage
   localStorage.setItem('pendingPaymentTxRef', txRef)
   ↓
4. Frontend redirects to Chapa
   ↓
5. User completes payment
   ↓
6. Chapa redirects to /payment/success (NO tx_ref in URL!)
   ↓
7. Frontend retrieves tx_ref from localStorage
   txRef = localStorage.getItem('pendingPaymentTxRef')
   ✅ NOW WE HAVE IT!
   ↓
8. Frontend verifies payment with backend
   ↓
9. Payment marked as COMPLETED
   ↓
10. Frontend clears localStorage
    ↓
11. Success page displays
