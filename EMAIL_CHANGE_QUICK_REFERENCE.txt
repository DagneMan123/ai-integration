🚀 HOW TO USE:

Step 1: Go to My Profile
   → Click "My Profile" in dashboard
   → Scroll to Email section

Step 2: Click "Change" Button
   → Email field becomes editable
   → New input appears

Step 3: Enter New Email
   → Type your new email address
   → Make sure it's correct

Step 4: Click "Confirm"
   → Email is validated
   → Email is updated
   → Success message appears

Step 5: Done!
   → Email is now updated
   → You can change it again anytime

📋 VALID EMAIL FORMAT:

✅ Valid Examples:
   • user@example.com
   • john.doe@company.co.uk
   • test+tag@domain.org
   • firstname.lastname@email.com

❌ Invalid Examples:
   • userexample.com (missing @)
   • user@example (missing extension)
   • user @example.com (space in email)
   • @example.com (missing username)

⚠️ VALIDATION RULES:

1. Email Format
   → Must be valid email format
   → Must contain @ symbol
   → Must have domain and extension

2. Duplicate Check
   → Cannot use same email as current
   → Must be different from existing email

3. Error Messages
   → "Please enter a valid email address"
   → "New email must be different from current email"
   → "Failed to update email"
