# Where AI is Used in Interview System

## 🤖 AI Usage Map

### 1. **Interview Question Generation**
**Location:** `server/services/aiService.js` → `generateInterviewQuestions()`

**What it does:**
- Analyzes job requirements and skills
- Generates 8-12 relevant, role-specific interview questions
- Adjusts difficulty based on experience level
- Creates questions tailored to the job position

**When it's used:**
- When employer creates a new interview
- Called from: `server/controllers/interviewController.js` → `startInterview()`

**Example:**
```javascript
// Interview starts → AI generates questions
const questions = await enhancedAI.generateInterviewQuestions(job, strictnessLevel, 10);
```

**Output:**
```json
[
  {
    "question": "Describe your experience with React hooks and explain when you would use useEffect vs useMemo",
    "type": "technical",
    "difficulty": "intermediate",
    "expectedKeywords": ["hooks", "lifecycle", "performance"]
  },
  {
    "question": "Tell me about a time you had to debug a complex issue. What was your approach?",
    "type": "behavioral",
    "difficulty": "intermediate"
  }
]
```

---

### 2. **Answer Evaluation & Scoring**
**Location:** `server/services/enhancedAIService.js` → `evaluateAnswer()`

**What it does:**
- Analyzes candidate's answer to each question
- Scores answer on relevance, completeness, accuracy, clarity
- Provides detailed feedback
- Generates score 0-100

**When it's used:**
- After candidate submits each answer
- Called from: `server/controllers/interviewController.js` → `submitAnswer()`

**Evaluation Criteria:**
- ✓ Relevance to question
- ✓ Completeness of answer
- ✓ Technical accuracy
- ✓ Clarity of explanation
- ✓ Use of examples
- ✓ Depth of understanding

**Example:**
```javascript
const evaluation = await enhancedAI.evaluateAnswer(
  question,
  candidateAnswer,
  job,
  strictnessLevel
);
// Returns: { score: 82, feedback: "Good understanding...", strengths: [...], weaknesses: [...] }
```

---

### 3. **AI Content Detection (Plagiarism Check)**
**Location:** `server/services/enhancedAIService.js` → `detectAIContent()`

**What it does:**
- Detects if answer was generated by ChatGPT, GPT-4, or similar AI tools
- Identifies plagiarized or copy-pasted content
- Analyzes writing patterns for AI signatures
- Provides confidence level (Low/Medium/High)

**When it's used:**
- After each answer submission
- Called from: `server/controllers/interviewController.js` → `submitAnswer()`

**Detection Methods:**
- Pattern analysis (AI writing style)
- Plagiarism database comparison
- Sentence structure analysis
- Vocabulary consistency check

**Example:**
```javascript
const plagiarismCheck = await enhancedAI.detectAIContent(candidateAnswer);
// Returns: {
//   isAIGenerated: false,
//   confidence: 0.15,
//   indicators: [],
//   recommendation: "Answer appears authentic"
// }
```

**Flags:**
- 🟢 Low (0-30%): Likely authentic
- 🟡 Medium (30-70%): Possible AI content
- 🔴 High (70-100%): Likely AI-generated

---

### 4. **Speech Analysis (Audio/Video Mode)**
**Location:** `server/services/enhancedAIService.js` → `analyzeSpeechPatterns()`

**What it does:**
- Analyzes speech rate and pace
- Counts filler words ("um", "uh", "like")
- Measures fluency and confidence
- Detects stress or nervousness indicators

**When it's used:**
- When interview mode is "audio" or "video"
- Called from: `server/controllers/interviewController.js` → `submitAnswer()`

**Metrics Measured:**
- Speech rate (words per minute)
- Filler word count
- Fluency score (0-100)
- Confidence indicators
- Pause patterns

**Example:**
```javascript
const speechAnalysis = enhancedAI.analyzeSpeechPatterns(audioTranscript, duration);
// Returns: {
//   speechRate: 145,
//   fillerCount: 3,
//   fluencyScore: 78,
//   confidenceLevel: "high"
// }
```

### 6. **Follow-Up Question Generation**
**Location:** `server/services/enhancedAIService.js` → `generateFollowUpQuestion()`

**What it does:**
- Analyzes candidate's answer
- Determines if follow-up is needed
- Generates contextual follow-up question
- Probes deeper into candidate knowledge

**When it's used:**
- After incomplete or unclear answers
- Called from: `server/controllers/interviewController.js` → `submitAnswer()`

**Triggers Follow-Up When:**
- Answer is incomplete
- Answer lacks specific examples
- Answer shows gaps in knowledge
- Answer needs clarification

**Example:**
```javascript

  originalQuestion,
  candidateAnswer,
  job
);
// Returns: {
//   followUp: "Can you provide a specific example of how you implemented this?",
//   reason: "Answer lacks concrete examples"
// }
```

### 7. **Comprehensive Report Generation**
**Location:** `server/services/enhancedAIService.js` → `generateComprehensiveReport()`

**What it does:**
- Analyzes all answers and metrics
- Generates detailed evaluation report
- Calculates multiple scores:

  - Communication score
  - Problem-solving score
  - Soft skills score
  - Confidence score
  - Fluency score
  - Professionalism score
  - Overall score

**When it's used:**
- When interview is completed
- Called from: `server/controllers/interviewController.js` → `completeInterview()`

**Report Includes:**
- Overall assessment
- Strengths and weaknesses
- Recommendations
- Detailed feedback per question
- Integrity analysis
- Hiring recommendation

**Example:**
```javascript
const report = await enhancedAI.generateComprehensiveReport(interview);
```

## 🌐 API Endpoints Using AI

### Interview Routes (`server/routes/interviews.js`)

```javascript
POST /api/interviews/start
├─ AI: Generate questions
└─ Returns: Interview with AI-generated questions

POST /api/interviews/:id/submit-answer
├─ AI: Evaluate answer
├─ AI: Detect AI content
├─ AI: Analyze sentiment
├─ AI: Generate follow-up
└─ Returns: Evaluation and next question

POST /api/interviews/:id/complete
├─ AI: Generate comprehensive report
├─ AI: Calculate all scores
└─ Returns: Final report with recommendations

GET /api/interviews/:id/report
└─ Returns: AI-generated interview report
```

## 🚀 How to Use AI Features

### For Developers:

**1. Generate Interview Questions:**
```javascript
const questions = await aiService.generateInterviewQuestions(jobDetails, 10);
```

**2. Evaluate Answer:**
```javascript
const evaluation = await enhancedAI.evaluateAnswer(question, answer, job, strictness);
```

**3. Detect AI Content:**
```javascript
const check = await enhancedAI.detectAIContent(answer);
```

**4. Generate Report:**
```javascript
const report = await enhancedAI.generateComprehensiveReport(interview);
```

### For Users:

**Candidates:**
- AI generates interview questions
- AI evaluates your answers

**Employers:**
- AI generates questions for you

- AI flags suspicious content
