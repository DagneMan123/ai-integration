IMPORTANT REMINDERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  PostgreSQL MUST be running before starting the server
    → Use START_POSTGRES_NOW.bat to start it automatically

⚠️  Keep both terminals (server and client) open while developing
    → Don't close them - they keep the services running

⚠️  If you close a terminal, restart that service
    → Server: cd server && npm run dev
    → Client: cd client && npm run dev

⚠️  Browser will auto-refresh when you make code changes
    → No need to manually refresh

⚠️  Check browser console for any errors
    → F12 to open developer tools


QUICK COMMANDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Start PostgreSQL:
  net start postgresql-x64-16

Stop PostgreSQL:
  net stop postgresql-x64-16

Start Server:
  cd server && npm run dev

Start Client:
  cd client && npm run dev

Test Database Connection:
  cd server
  node -e "const {PrismaClient} = require('@prisma/client'); const p = new PrismaClient(); p.$connect().then(() => console.log('✅ Connected')).catch(e => console.log('❌', e.message))"

View Server Logs:
  tail -f server/logs/combined.log

Clear npm Cache:
  npm cache clean --force

Reinstall Dependencies:
  cd server && npm install
  cd client && npm install


TROUBLESHOOTING QUICK REFERENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Can't reach database server at localhost:5432"
   → PostgreSQL is not running
   → Solution: Run START_POSTGRES_NOW.bat

❌ "Port 5000 already in use"
   → Another process is using port 5000
   → Solution: taskkill /F /IM node.exe

❌ "Port 3000 already in use"
   → Another process is using port 3000
   → Solution: taskkill /F /IM node.exe

❌ "Module not found" errors
   → Dependencies not installed
   → Solution: npm install (in both server and client)

❌ "Chatbot not responding"
   → Check OpenAI API key in server/.env
   → Verify server is running on port 5000
   → Check browser console for errors

❌ "Database connection failed"
   → PostgreSQL not running
   → Database doesn't exist
   → Solution: Run npx prisma migrate dev
