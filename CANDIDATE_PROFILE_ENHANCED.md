## Profile Fields

## How to Use

### 1. Access Your Profile
- Log in as a candidate
- Navigate to "My Profile" from the dashboard
- Your current profile information will load automatically

### 2. Update Your Information
- Edit any field (except email)
- The save button will enable when you make changes
- Fill in all relevant information to improve job matching

### 3. Save Changes
- Click "Save Changes" button
- Wait for the success message
- Your profile is now updated

### 4. Tips for Best Results

**Professional Bio**
- Keep it concise (2-3 sentences)
- Highlight your key strengths
- Mention your career goals
- Example: "Experienced full-stack developer with 5+ years in web development. Passionate about building scalable applications and mentoring junior developers."

**Skills**
- List relevant technical skills
- Separate with commas
- Include both hard and soft skills
- Example: "JavaScript, React, Node.js, TypeScript, Problem Solving, Team Leadership"

**Work Experience**
- Include job titles, companies, and dates
- Describe key responsibilities and achievements
- Use bullet points for clarity
- Quantify achievements when possible

**Education**
- List degrees and certifications
- Include institution names and graduation dates
- Add relevant coursework or honors
- Include online certifications

## Features Explained

### Change Tracking
The save button is disabled until you make changes to the form. This prevents accidental saves and makes it clear when your profile has been modified.

**How it works:**
- Button is gray and disabled when no changes
- Button becomes blue and enabled when you edit a field
- After saving, button returns to disabled state
- Message appears: "Make changes to enable the save button"

### Success Feedback
When you save your profile, you'll see:
1. Green success notification at the top
2. Toast message in the corner
3. Auto-dismisses after 3 seconds
4. Save button returns to disabled state

### Error Handling
If something goes wrong:
1. Error message appears in red
2. Toast notification shows the error
3. You can try saving again
4. Check your internet connection if errors persist

### Loading State
When saving:
1. Button shows "Saving Changes..." text
2. Loading spinner appears
3. Button is disabled during save
4. Prevents duplicate submissions

## Troubleshooting

### Issue: Save button is disabled
**Solution**: Make sure you've made changes to at least one field. The button only enables when changes are detected.

### Issue: Error message appears
**Solution**: Check your internet connection and try again. If the error persists, contact support.

### Issue: Changes not saved
**Solution**: Make sure you clicked the "Save Changes" button and waited for the success message.

### Issue: Profile information not loading
**Solution**: Refresh the page or log out and log back in.

### Issue: Can't edit email
**Solution**: Email cannot be changed from the profile page. Contact support if you need to change your email.
