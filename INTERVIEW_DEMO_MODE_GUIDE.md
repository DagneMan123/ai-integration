# Interview Demo Mode Guide

## What is Demo Mode?

Demo Mode allows you to practice interviews without requiring camera and microphone permissions. It's perfect for:
- Testing the interview workflow
- Practicing without hardware
- Learning how the system works
- Troubleshooting permission issues

## How to Enable Demo Mode

### Step 1: Start Practice
1. Navigate to the Practice page
2. Select a practice type (Technical, Behavioral, or Case Study)
3. Click "Start Practice"

### Step 2: Hardware Check Screen
When you see the hardware check screen with errors:
1. You'll see two buttons: "Retry" and "Demo Mode"
2. Click the "Demo Mode" button (purple button with lightning bolt icon)

### Step 3: Demo Mode Activated
- Camera status will show "Demo" (purple)
- Microphone status will show "Demo" (purple)
- Microphone volume will show a simulated level
- "Begin Interview" button will become enabled

### Step 4: Start Interview
1. Click "Begin Interview"
2. The interview environment will load with a purple demo placeholder instead of camera feed
3. You can now practice recording and answering questions

## Demo Mode Features

### What Works in Demo Mode
✅ Full 4-phase interview workflow
✅ Question display and navigation
✅ Recording simulation (no actual video recorded)
✅ Timer countdown
✅ Progress tracking
✅ Response submission
✅ Results and feedback display
✅ All UI interactions

### What's Different in Demo Mode
- Camera feed shows purple placeholder with "Demo Mode" indicator
- Microphone volume is simulated (random values)
- Recordings are not actually saved (mock blobs created)
- No actual video/audio data is captured

## Using Demo Mode

### Recording Answers
1. Click "Start Recording" - recording will begin (no actual capture)
2. Answer the question out loud (for practice)
3. Click "Stop Recording" when done
4. Click "Submit & Next" to move to the next question

### Navigating Questions
- Each question displays normally
- Progress bar shows your position
- Timer counts down as usual
- You can answer all questions or let timer auto-submit

### Viewing Results
- After completing all questions, results page displays
- AI feedback is simulated
- Scores are generated (random 70-100 range)
- You can download report or try again

## When to Use Demo Mode

### Use Demo Mode When:
- Camera/microphone permissions are denied
- You don't have hardware available
- You want to test the workflow
- You're learning how the system works
- You want to practice without recording

### Use Real Mode When:
- You have camera and microphone available
- You've granted permissions
- You want actual recordings for feedback
- You're doing a real practice session

## Switching Between Modes

### From Demo to Real
1. Complete demo session
2. Click "Try Again"
3. Click "Retry" on hardware check
4. Grant camera/microphone permissions
5. Begin real interview

### From Real to Demo
1. If hardware fails, you'll see error screen
2. Click "Demo Mode" button
3. Continue with demo mode

## Troubleshooting Demo Mode

### Demo Mode Button Not Showing
- Make sure you're on the hardware check screen
- Make sure permissions are denied (not just checking)
- Refresh the page and try again

### Demo Mode Not Working
- Clear browser cache
- Refresh the page
- Try a different browser
- Check browser console for errors

### Can't Submit Responses in Demo
- Make sure you clicked "Start Recording" first
- Make sure you clicked "Stop Recording"
- Then click "Submit & Next"
- If still not working, refresh page

## Demo Mode Limitations

1. **No Actual Recording**: Videos are not saved
2. **Simulated Feedback**: AI feedback is randomly generated
3. **No Persistence**: Session data is not stored
4. **Simulated Scores**: Scores are random, not based on performance
5. **No Real Analysis**: No actual AI analysis of responses

## Switching to Real Mode

To use the real interview system with actual recordings:

1. **Grant Permissions**:
   - Click lock icon in address bar
   - Find Camera and Microphone
   - Change from "Block" to "Allow"
   - Refresh page

2. **Retry Hardware Check**:
   - Click "Retry" button
   - System will request permissions again
   - Grant access when prompted

3. **Begin Real Interview**:
   - "Begin Interview" button will enable
   - Click to start real interview
   - Actual video/audio will be recorded

## Tips for Demo Mode

### Best Practices
1. Use demo mode to learn the workflow first
2. Practice your answers out loud (even without recording)
3. Pay attention to the timer and pacing
4. Review the feedback to understand scoring
5. Then switch to real mode for actual practice

### Practice Tips

- Look at the camera placeholder

- Take time to think before answering
- Practice multiple times to improve

## FAQ

**Q: Will my demo session be saved?**
A: No, demo sessions are not saved. They're for practice only.

**Q: Can I convert demo to real recording?**
A: No, you need to do a real session with actual hardware.

**Q: Is demo mode cheating?**
A: No, it's a practice tool. Real interviews require actual hardware.

**Q: Can I use demo mode in production?**
A: Demo mode is for testing and learning. Real interviews require real hardware.

**Q: How do I know if I'm in demo mode?**
A: Look for the purple "Demo Mode" indicator on the camera feed.

**Q: Can I switch to real mode mid-interview?**
A: No, you need to complete or cancel the session first.

**Q: Is the feedback accurate in demo mode?**
A: No, feedback is simulated. Real feedback requires actual analysis.
