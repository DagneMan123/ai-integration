✅ PAYMENT FLOW:

1. User clicks "Subscribe Now"
2. Frontend calls POST /api/payments/initialize
3. Backend generates unique tx_ref
4. Backend calls Chapa API (with SECRET_KEY)
5. Backend returns checkout_url
6. Frontend redirects to Chapa
7. User completes payment
8. Chapa redirects to /payment/success?tx_ref=XXX&status=success
9. Frontend captures tx_ref
10. Frontend calls POST /api/payments/verify/:tx_ref
11. Backend verifies with Chapa (server-to-server)
12. Backend calls OpenAI (only after verification)
13. Frontend displays success message
