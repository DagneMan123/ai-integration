❌ PROBLEM:

Error: Invalid `prisma.candidateProfile.update()` invocation
Unknown argument `bio`

This error occurred when trying to update profile with:
• bio
• email
• phone

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 ROOT CAUSE:

The code was trying to update candidateProfile table with fields that
don't exist there. These fields exist in the User table instead:

❌ Wrong:
  candidateProfile.update({
    bio: "...",      // Not in candidateProfile
    email: "...",    // Not in candidateProfile
    phone: "..."     // Not in candidateProfile
  })

✅ Correct:
  user.update({
    bio: "...",      // In User table
    email: "...",    // In User table
    phone: "..."     // In User table
  })

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ SOLUTION:

Updated server/controllers/userController.js to:

1. Separate User fields from CandidateProfile fields
2. Update User table with: firstName, lastName, email, phone, bio
3. Update CandidateProfile table with: skills, experience, education
