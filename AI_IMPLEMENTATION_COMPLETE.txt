┌────────────────────────────────────────────────────────────────────────────┐
│                          TROUBLESHOOTING                                   │
└────────────────────────────────────────────────────────────────────────────┘

Problem: "AI service is not available"
Solution:
  1. Check OpenAI API key in server/.env
  2. Verify API key is valid
  3. Check OpenAI account has credits
  4. Restart server

Problem: "Failed to generate questions"
Solution:
  1. Ensure job details are complete
  2. Check OpenAI API quota
  3. Try with fewer questions
  4. Check server logs

Problem: "Cannot connect to database"
Solution:
  1. Start PostgreSQL: net start postgresql-x64-16
  2. Check database exists: simuai_db
  3. Run migrations: npx prisma db push

Problem: "401 Unauthorized"
Solution:
  1. Login first
  2. Check token is valid
  3. Refresh page and try again
