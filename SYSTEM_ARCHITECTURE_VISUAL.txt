INTERVIEW PROCESS FLOW:

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  1. CANDIDATE STARTS INTERVIEW                                             │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Frontend                                                         │   │
│     │ POST /api/interviews/start                                      │   │
│     │ { jobId, applicationId }                                        │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Backend (interviewController.startInterview)                    │   │
│     │ • Verify job exists                                             │   │
│     │ • Verify application exists                                     │   │
│     │ • Call enhancedAI.generateInterviewQuestions()                  │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ AI Service (enhancedAIService.generateInterviewQuestions)       │   │
│     │ • Analyze job description                                       │   │
│     │ • Call OpenAI API                                               │   │
│     │ • Generate 10 questions                                         │   │
│     │ • Return questions array                                        │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Database (Prisma)                                               │   │
│     │ prisma.interview.create({                                       │   │
│     │   jobId, candidateId, questions, status: 'IN_PROGRESS'         │   │
│     │ })                                                              │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Frontend                                                         │   │
│     │ Display questions to candidate                                  │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  2. CANDIDATE ANSWERS QUESTION                                             │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Frontend                                                         │   │
│     │ POST /api/interviews/:id/submit-answer                          │   │
│     │ { questionIndex, answer, timeTaken }                            │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Backend (interviewController.submitAnswer)                      │   │
│     │ • Get interview from database                                   │   │
│     │ • Call enhancedAI.evaluateAnswer()                              │   │
│     │ • Call enhancedAI.detectAIContent()                             │   │
│     │ • Call enhancedAI.analyzeSentiment()                            │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ AI Service (enhancedAIService.evaluateAnswer)                   │   │
│     │ • Analyze candidate answer                                      │   │
│     │ • Compare with expected keywords                                │   │
│     │ • Call OpenAI API for evaluation                                │   │
│     │ • Generate score (0-100)                                        │   │
│     │ • Generate feedback                                             │   │
│     │ • Return evaluation                                             │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Database (Prisma)                                               │   │
│     │ prisma.interview.update({                                       │   │
│     │   responses: [..., { answer, score, feedback }]                │   │
│     │ })                                                              │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Frontend                                                         │   │
│     │ Display score, feedback, next question                          │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  3. INTERVIEW COMPLETES                                                    │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Frontend                                                         │   │
│     │ POST /api/interviews/:id/complete                               │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Backend (interviewController.completeInterview)                 │   │
│     │ • Get interview from database                                   │   │
│     │ • Call enhancedAI.generateComprehensiveReport()                 │   │
│     │ • Calculate integrity score                                     │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │

│     │ AI Service (enhancedAIService.generateComprehensiveReport)      │   │

│     │ • Calculate technical score                                     │   │

│     │ • Calculate soft skills score                                   │   │
│     │ • Generate recommendations                                      │   │
│     │ • Return comprehensive report                                   │   │
│     └──────────────────────────────────────────────────────────────────┘   │
│                              ↓                                              │
│     ┌──────────────────────────────────────────────────────────────────┐   │
│     │ Database (Prisma)                                               │   │
│     │ prisma.interview.update({                                       │   │

═══════════════════════════════════════════════════════════════════════════════
                          COMPONENT INTERACTIONS
═══════════════════════════════════════════════════════════════════════════════

CANDIDATE DASHBOARD ↔ DATABASE
  ├─ GET /api/interviews → Fetch all interviews
  ├─ GET /api/applications → Fetch all applications
  ├─ GET /api/users/:id → Fetch profile
  └─ PUT /api/users/:id → Update profile

EMPLOYER DASHBOARD ↔ DATABASE
  ├─ GET /api/jobs → Fetch all jobs
  ├─ POST /api/jobs → Create job
  ├─ GET /api/jobs/:id/candidates → Fetch job candidates
  └─ GET /api/analytics → Fetch analytics

ADMIN DASHBOARD ↔ DATABASE
  ├─ GET /api/admin/users → Fetch all users
  ├─ GET /api/admin/jobs → Fetch all jobs
  ├─ GET /api/admin/payments → Fetch all payments
  └─ GET /api/admin/analytics → Fetch analytics

INTERVIEW FLOW ↔ AI SERVICE ↔ DATABASE
  ├─ POST /api/interviews/start → Generate questions (AI) → Store in DB
  ├─ POST /api/interviews/:id/submit-answer → Evaluate (AI) → Store in DB
  ├─ POST /api/interviews/:id/complete → Generate report (AI) → Store in DB
  └─ GET /api/interviews/:id/report → Fetch from DB → Display
