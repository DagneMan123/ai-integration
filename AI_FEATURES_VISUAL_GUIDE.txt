┌────────────────────────────────────────────────────────────────────────────┐
│                          HOW INTERVIEW WORKS                               │
└────────────────────────────────────────────────────────────────────────────┘

INTERVIEW FLOW:

  Employer Creates Job
         ↓
  Candidate Applies
         ↓
  Candidate Starts Interview
         ↓
  AI Generates Questions (2-5 sec)
         ↓
  Candidate Sees Question 1
         ↓
  Candidate Types Answer
         ↓
  Candidate Clicks "Submit"
         ↓
  AI Evaluates Answer (3-8 sec)
         ↓
  Candidate Sees Score & Feedback
         ↓
  Candidate Sees Question 2
         ↓
  ... (repeat for all questions)
         ↓
  Candidate Clicks "Complete"
         ↓
  AI Generates Report (5-10 sec)
         ↓
  Candidate Views Report
         ↓
  Report Shows:
  • Overall Score
  • Technical Score
  • Communication Score
  • Problem-Solving Score
  • Strengths
  • Weaknesses
  • Recommendations

┌────────────────────────────────────────────────────────────────────────────┐
│                          TROUBLESHOOTING                                   │
└────────────────────────────────────────────────────────────────────────────┘

Problem: "AI service is not available"
Solution:
  1. Check OpenAI API key in server/.env
  2. Add credits to OpenAI account
  3. Wait 5 minutes
  4. Restart server

Problem: "Failed to generate questions"
Solution:
  1. Ensure job details are complete
  2. Check job title is provided
  3. Try with fewer questions
  4. Check server logs

Problem: "401 Unauthorized"
Solution:
  1. Login first
  2. Check token is valid
  3. Refresh page

Problem: "Cannot connect to database"
Solution:
  1. Start PostgreSQL: net start postgresql-x64-16
  2. Check database exists
  3. Run migrations

Problem: "Sidebar is cut off"
Solution:
  1. Already fixed!
  2. Refresh page to see changes
