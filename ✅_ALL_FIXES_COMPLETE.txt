🔍 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Error: "Can't reach database server at localhost:5432"
→ PostgreSQL is not running
→ Start PostgreSQL service via Windows Services

Error: "Route.get() requires a callback function"
→ Already fixed - restart backend server

Error: "Loading chunk failed (timeout)"
→ Clear client build cache:
  cd client && rmdir /s /q build && npm install

Error: "Port 5000 already in use"
→ Kill process on port 5000:
  netstat -ano | findstr :5000
  taskkill /PID <PID> /F

Error: "npm: command not found"
→ Install Node.js from https://nodejs.org/
