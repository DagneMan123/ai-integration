   - "Save Changes" button enables/disables based on hasChanges
   - Status message shows "● You have unsaved changes" or "✓ Everything is up to date"

   - Profile completion percentage updates in real-time

═══════════════════════════════════════════════════════════════════════════════
                        WHAT CHANGED
═══════════════════════════════════════════════════════════════════════════════

File: client/src/pages/candidate/Profile.tsx

1. Added formData state:
   const [formData, setFormData] = useState<ProfileFormData | null>(null);

2. Added real-time watch effect:
   useEffect(() => {
     const subscription = watch((data) => {
       setFormData(data as ProfileFormData);
       setHasChanges(true);
     });
     return () => subscription.unsubscribe();
   }, [watch]);

3. Updated all form fields:
   - Removed onChange from register()
   - Added onChange={() => setHasChanges(true)} to inputs
   - This ensures proper state management
