## 🎯 How to Identify AI Features

### Look for These Indicators:

**1. "AI-Powered" Label**
- Indicates AI is analyzing/generating content
- Usually appears in headers or badges

**2. "Generated by AI" Badge**
- Shows content was created by AI
- Appears on questions, recommendations, reports

**3. Confidence Scores**
- Shows AI confidence level
- Usually 0-100% or Low/Medium/High

**4. "Learn More" Links**
- Explains how AI feature works
- Provides transparency

**5. Timestamp**
- Shows when AI analysis was performed
- Helps track analysis recency

## 🔍 Finding AI in Code

### Frontend (React/TypeScript)

**Look for these patterns:**

```typescript
// AI Icon imports
import { FiZap, FiShield, FiTrendingUp } from 'react-icons/fi';

// AI Badge components
<span className="ai-badge">🤖 AI-Powered</span>

// AI Score display
<div className="ai-score">
  <span>AI Score: {score}</span>
  <span className="confidence">{confidence}%</span>
</div>

// AI Alert
<div className="ai-alert">
  <FiAlertCircle /> AI detected possible plagiarism
</div>
```

### Backend (Node.js/JavaScript)

**Look for these patterns:**

```javascript
// AI Service calls
const evaluation = await aiService.evaluateAnswer(answer);
const report = await enhancedAI.generateComprehensiveReport(interview);
const check = await enhancedAI.detectAIContent(answer);

// AI Response handling
if (plagiarismCheck.isAIGenerated) {
  // Flag as suspicious
}

// AI Scoring
const score = evaluation.score; // 0-100
const confidence = evaluation.confidence; // 0-1
```
