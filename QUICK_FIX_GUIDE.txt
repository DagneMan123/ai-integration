TROUBLESHOOTING:
================

Problem: "net start postgresql-x64-16" doesn't work
Solution: Check your PostgreSQL version
         Type: sc query | findstr postgresql
         Use the service name you see

Problem: "psql: command not found"
Solution: PostgreSQL is not installed or not in PATH
         Install PostgreSQL from: https://www.postgresql.org/download/

Problem: "password authentication failed"
Solution: Check password in server/.env file
         Should be: DATABASE_URL="postgresql://postgres:MYlove8@localhost:5432/simuai_db?schema=public"

Problem: "Port 5432 is already in use"
Solution: Another PostgreSQL is running
         Type: netstat -ano | findstr :5432
         Kill the process or use different port
