# Quick Start - Practice Interview

## ✅ Status: Ready to Use

## Start Servers

### Terminal 1 - Backend
```bash
cd server
npm run dev
```

### Terminal 2 - Frontend
```bash
cd client
npm start
```

## Access Application

1. Open http://localhost:3000
2. Log in as candidate
3. Go to **Practice** section
4. Click **Begin Interview**

## Use Practice Interview

1. **Read** the question on screen
2. **Record** your answer (click microphone)
3. **Stop** recording (click stop button)
4. **Submit** response (click submit button)
5. **Next** question (auto-advances)
6. **Repeat** for all questions
7. **Complete** interview

## Troubleshooting

**Backend won't start**
- Check if port 5000 is in use
- Kill process or use different port

**Frontend won't start**
- Clear cache: `npm cache clean --force`
- Reinstall: `npm install`

**Camera not working**
- Check browser permissions
- Allow camera access
- Try different browser

**Video upload fails**
- Check internet connection
- Check Cloudinary credentials
- Check browser console
