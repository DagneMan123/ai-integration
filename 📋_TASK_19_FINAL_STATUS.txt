═══════════════════════════════════════════════════════════════════════════════
                        TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ISSUE: "Can't reach database server at localhost:5432"
CAUSE: PostgreSQL is not running
SOLUTION: START_POSTGRESQL_WINDOWS.bat

ISSUE: "Insufficient credits" error still appears
CAUSE: Database wasn't reseeded
SOLUTION: cd server && npx prisma db seed

ISSUE: Wallet still shows 0 credits after reseeding
CAUSE: Browser cache or session not refreshed
SOLUTION: Clear cache (Ctrl+Shift+Delete) and login again

ISSUE: Seed command fails with "Prisma Client not found"
CAUSE: Dependencies not installed
SOLUTION: cd server && npm install && npx prisma db seed

ISSUE: "Payment modal shows 'Pay & Start Interview' instead of 'Start Interview Now'"
CAUSE: Wallet balance not properly fetched or still 0
SOLUTION: Verify reseeding completed, check wallet in database, reseed if needed
