TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────────

Issue: Payment modal doesn't appear
  → Check: Browser console for errors
  → Check: Backend logs for errors
  → Check: localStorage has 'pendingInterviewId'
  → Solution: Refresh page and try again

Issue: Chapa page doesn't load
  → Check: CHAPA_API_KEY in server/.env
  → Check: FRONTEND_URL in server/.env
  → Check: Network tab for API errors
  → Solution: Check backend logs for Chapa API errors

Issue: Payment verification fails
  → Check: Backend logs for Chapa API errors
  → Check: Network timeout (should be 60 seconds)
  → Check: Payment record in database
  → Solution: Retry payment verification

Issue: Interview page doesn't load
  → Check: Browser console for errors
  → Check: paymentVerified=true in URL
  → Check: Backend logs for errors
  → Solution: Refresh page and try again

Issue: "Payment Required" error on interview
  → Cause: paymentVerified parameter missing
  → Solution: Complete payment flow again
